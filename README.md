### About Me

I'm a cybersecurity professional with **7+ years of combined experience** spanning security-focused software QA, secure SDLC, SOC operations, detection engineering, incident response, vulnerability management, cloud security, and AI/LLM security.

My career started in **security-focused software QA**, where I spent years validating authentication flows, access control boundaries, input handling, and OWASP Top 10 vulnerability classes across full software development lifecycles. That foundation in *finding security issues before they ship* shaped how I approach security operations today — defensively, methodically, and with a deep appreciation for the SDLC.

From there I moved into **dedicated cybersecurity** as a lead analyst running SOC operations across 300+ client environments at an MSP, managing detection engineering, IR, vulnerability management, and AI tool security evaluation across regulated industries.

Currently pursuing my **M.S. Cybersecurity at Georgia Tech**, focusing on AI security, applied security analytics, and DFIR.

I build because security tooling shouldn't be locked behind enterprise paywalls. Every tool here is open-source, MIT licensed, and built with production-grade security practices.
